# # Expert Pro Dicas para DIY

É um clichê por uma razão. Verifique sempre as suas medições uma última vez antes de fazer quaisquer alterações ou cortes permanentes.
- **Preparação de superfície**: 70% de uma boa pintura é o trabalho de preparação. Sanding, limpeza, e priming são mais importantes do que o estágio real da pintura.
- **Compre o básico da qualidade**: Você não precisa de uma ferramenta para tudo, mas compre o melhor que puder para aqueles que você mais usar (drill, martelo, fita métrica).


# # Erros comuns para evitar

- **Skipping Safety Gear**: A proteção dos olhos e ouvidos não é opcional. A maioria das lesões de DIY são evitáveis com um simples par de óculos de segurança.
- **Começando de forma Muito Grande**: Não renove toda a sua cozinha como um primeiro projeto. Comece com vitórias menores como construção de prateleiras para construir sua confiança e habilidades.


# # Perguntas frequentes

Vale a pena reparar ou substituir?
Se o custo de reparo é mais de 50% do custo de substituição, geralmente é melhor atualizar. Se for um item 'clássico', o reparo é quase sempre melhor.

Como encontro um garanhão sem um garanhão?
Procure tomadas (geralmente ligadas aos pregos), toque na parede e ouça um som sólido, ou use um ímã forte para encontrar os parafusos drywall.
